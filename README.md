# MW Petroleum Real Options Valuation

## Overview

This project evaluates the acquisition of MW Petroleum by Apache Corporation using multiple valuation frameworks, including Discounted Cash Flow (DCF), Adjusted Present Value (APV), and Black-Scholes real options analysis.

Unlike traditional valuation exercises that rely solely on projected operating cash flows, this analysis incorporates managerial flexibility in reserve development decisions. Because oil and gas investments are highly sensitive to commodity price volatility and involve substantial irreversible capital expenditures, the ability to delay development creates meaningful economic value beyond a static DCF estimate.

The project examines whether Apache Corporation should acquire MW Petroleum and how embedded real options affect the firm's intrinsic value.

---

## Investment Question

**How much value does managerial flexibility add to MW Petroleum beyond its traditional DCF valuation?**

The analysis focuses on:

* Valuing MW Petroleum's reserve portfolio
* Estimating debt capacity and financing benefits
* Quantifying the value of delaying reserve development
* Evaluating acquisition attractiveness under commodity price uncertainty

---

## Key Findings

### 1. Traditional DCF Valuation

Using a discounted cash flow framework and a 14.18% unlevered cost of capital, MW Petroleum's reserve portfolio was valued at approximately:

| Metric               | Value    |
| -------------------- | -------- |
| DCF Enterprise Value | ~$483.7M |

The valuation incorporates:

* Proved developed reserves
* Proved undeveloped reserves
* Probable reserves
* Possible reserves

### 2. Capital Structure and APV Analysis

An Adjusted Present Value (APV) framework was used to separate:

* Operating value
* Financing value from tax shields

Debt capacity was constrained to 50% of the value of proved reserves, reflecting lending practices within the oil and gas industry.

### 3. Real Options Create Additional Value

The most significant source of incremental value arises from management's ability to delay development of undeveloped reserves.

Using the Black-Scholes framework:

| Reserve Category   | Option Value |
| ------------------ | ------------ |
| Proved Undeveloped | $48.33M      |
| Probable           | $49.17M      |
| Possible           | $37.57M      |

The analysis demonstrates that flexibility to defer investment increases value beyond what is captured in a traditional DCF model.

### 4. Waiting Has Economic Value

Higher commodity price volatility increases the value of waiting before committing capital.

Because reserve development is largely irreversible, Apache benefits from preserving flexibility and exercising development options only when market conditions become favorable.

---

## Methodology

### Discounted Cash Flow (DCF)

Reserve categories were valued using projected operating cash flows and terminal values:

* Proved Developed Reserves
* Proved Undeveloped Reserves
* Probable Reserves
* Possible Reserves

All projected cash flows were discounted using a risk-adjusted unlevered cost of capital.

### Adjusted Present Value (APV)

The APV framework separates:

1. Unlevered operating value
2. Present value of financing benefits

Debt capacity was modeled using reserve-backed lending constraints commonly applied within the energy industry.

### Real Options Analysis

Real options were modeled using the Black-Scholes framework.

Key inputs included:

| Input              | Value   |
| ------------------ | ------- |
| Risk-Free Rate     | 8.03%   |
| Volatility         | 40%     |
| Time to Expiration | 7 Years |

The option value reflects management's ability to delay reserve development until market conditions improve.

## Conclusion

The valuation indicates that MW Petroleum's value extends beyond its projected operating cash flows. The ability to delay reserve development creates substantial economic value in a volatile commodity environment, making real-options analysis a critical complement to traditional valuation techniques.

The results suggest that Apache Corporation can justify the acquisition not only through projected cash flows, but also through the strategic flexibility embedded within MW Petroleum's undeveloped reserve portfolio.
